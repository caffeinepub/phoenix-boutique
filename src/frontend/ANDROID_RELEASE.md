# Android Release APK Build Guide

This guide explains how to build a release APK for Phoenix Boutique that can be distributed and installed on Android devices.

## Prerequisites

Before building the release APK, ensure you have:

1. **Node.js and npm/pnpm** installed (for building the web assets)
2. **Android Studio** or **Android SDK Command Line Tools** installed
3. **Java Development Kit (JDK)** 8 or higher
4. **Capacitor CLI** installed globally: `npm install -g @capacitor/cli`
