# Phoenix Boutique - Offline-First Order Management

A luxury boutique order management application with offline-first architecture and optional Firebase cloud sync.

## Features

- **Offline-First**: Full functionality without internet connection
- **Order Management**: Create, edit, and track customer orders
- **Payment Tracking**: Record payments with history and audit trails
- **Financial Reports**: Dashboard metrics and audit summaries
- **CSV Export**: Export orders for external analysis
- **Printable Views**: Print-friendly order summaries
- **Firebase Integration** (Optional):
  - Email/Password authentication
  - Cloud data synchronization
  - Image storage in Firebase Storage
  - Multi-device access
