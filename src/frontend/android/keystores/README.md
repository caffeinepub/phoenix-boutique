# Android Keystores Directory

This directory is designated for storing Android signing keystores used to sign release builds of Phoenix Boutique.

## ⚠️ CRITICAL SECURITY WARNING

**DO NOT commit keystores or passwords to version control.**

All `.keystore`, `.jks`, and `keystore.properties` files in this directory are (and must remain) gitignored.
